## Projects

### System Health Monitor
A CLI tool to monitor Linux server health.

**Usage:**
```bash
./scripts/health-monitor.sh --report    # Full report
./scripts/health-monitor.sh --cpu       # CPU only
./scripts/health-monitor.sh --memory    # Memory only
./scripts/health-monitor.sh --disk      # Disk only
./scripts/health-monitor.sh --network   # Network only
./scripts/health-monitor.sh --help      # Help
```

**Features:**
- CPU, memory, disk usage monitoring
- Service status checks
- Network connectivity check
- Color coded warnings and alerts
- Modular CLI with flags
